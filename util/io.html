magna.util.io.untar(file_path, dir_name)

Extracts the contents of the tar file into the target directory.

Parameters

  • file_path (str) – The path to the tar file.

  • dir_name (str) – The directory to extract the tar file into.

Examples

Extract the contents of the tar file at /tmp/data.tar.gz into the directory /tmp/data.

>>> untar('/tmp/data.tar.gz', '/tmp/data')

magna.util.io.download_file(url, path, md5=None)

Downloads a file to disk, optionally validating the md5 hash.

Parameters

  • url (str) – The url to download from.

  • path (str) – The path to save the file to.

  • md5 (Optional[str]) – The expected md5 hash of the file.

Raises

IOError – If the md5 hash doesn’t match.

Examples

Download the file at https://www.example.com/data.csv to /tmp/data.csv.

>>> download_file('https://www.example.com/data.csv', '/tmp/data.csv')
magna.util.io.cache_file(srv_path, local_name)

Copies a file to the magna cache (doesn’t auto-remove).

Parameters

  • local_name (str) – The key to cache this file with.

Return type

Returns

Examples
