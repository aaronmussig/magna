magna.util.disk.untar(file_path, dir_name)

Extracts the contents of the tar file into the target directory.

Parameters

  • file_path (str) – The path to the tar file.

  • dir_name (str) – The directory to extract the tar file into.

Examples

Extract the contents of the tar file at /tmp/data.tar.gz into the directory /tmp/data.

>>> untar('/tmp/data.tar.gz', '/tmp/data')

magna.util.disk.cache_file(srv_path, local_name)

Copies a file to the magna cache (doesn’t auto-remove).

Parameters

  • local_name (str) – The key to cache this file with.

Return type

str

Returns

Examples

Cache the file at /srv/data.csv as data.csv.

>>> cache_file('/srv/data.csv', 'data.csv')
